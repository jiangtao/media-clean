# 定期清理准备执行计划

## Goal

在现有 Android 优先 MVP 上补齐“为定期清理做准备”的最小可交付能力：允许用户开启本地提醒、设置提醒频率与时间、持久化设置，并在界面中清晰展示当前提醒状态。

## Scope

1. 增加本地提醒设置模型与持久化。
2. 使用 Expo 本地通知安排每日或每周提醒。
3. 在首页展示提醒状态卡片与快捷配置入口。
4. 补充必要的验证与文档说明。
