# 扫描入口交互优化执行计划

## 目标

依据 [扫描入口交互优化设计](../2026-04-18-scan-entry-interaction-design/_index.md)，完成照片页扫描入口的页内交互重构，并将规则同步写入交互标准。

## 任务

1. 交互标准与设计文档固化
   - 新增独立交互标准文档
   - 记录扫描入口标准
2. 扫描反馈组件重构
   - 将 `ScanProgress` 从弹窗改为页内内嵌状态轨道
   - 支持扫描中与完成态摘要
3. 照片页入口重构
   - 缩小授权/扫描文案层级
   - 展示默认扫描范围摘要
   - 接入内嵌扫描轨道
4. 测试与验收
   - 更新页面与组件测试
   - 运行类型检查与全量测试

## 验收标准

1. 不再出现扫描进度弹窗。
2. 扫描中显示页内流水线进度与 `current/total`。
3. 扫描完成后原位显示异常媒体数量。
4. 交互标准文档已更新。
5. `npm run typecheck -- --pretty false` 与 `npm run test -- --run` 通过。
