## 新增执行入口

### Emulator / AVD 设计签收

```bash
npm run verify:android:v0-4-design-signoff
```

该入口默认不传 `--install-apk`，会串起 current-size light、`scan-complete`、SE light、SE dark、display cutout overlay 和 Maestro smoke。

### 物理 Android 真机签收

```bash
npm run verify:android:v0-4-physical-signoff -- --serial <device-serial>
```

该入口要求目标必须是真机/非 emulator；如果误连 emulator 会直接失败。dry-run 已验证默认 emulator 分支会被 `--require-physical` 拒绝。它不做 emulator 的 SE / cutout override，也默认跳过 Maestro；真机/非 emulator 会跑 current-size light、`scan-complete` 和 current-size dark，并在退出时恢复主题。
