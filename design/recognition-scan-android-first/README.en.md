# Android-First Scan And Recognition Design

中文版本: [README.md](./README.md)

This directory answers one question: **what the Android target state for scanning and recognition should be.**

Reading order:

1. [Target-State Goals](./target-state-goals.en.md)
2. [Algorithm Research](./algorithm-research.en.md)
3. [Android-First Scan And Recognition Architecture](./architecture.en.md)

Current constraints:

- Clarify Android first, without blending in iOS, CLI, or skill abstractions.
- Enumeration must read media dimensions, duration, file size, and related metadata.
- Recognition dimensions must map to established algorithms or explicit AI model choices.
- Goals define what must be true; architecture defines how to make it true.
