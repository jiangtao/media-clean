# 最佳实践 - iOS 风格 UI 重新设计

## 1. iOS 设计模式

### 1.1 Tab Bar

| 方面 | 指南 |
|------|------|
| **位置** | 屏幕底部（iPhone 上永不在顶部） |
| **最大 tabs** | 5 个（6+ 项进入 "更多" 菜单） |
| **推荐数量** | 3-5 个 tabs 以获得最佳可用性 |
| **图标大小** | 25×25 pt（常规），18.5×18.5 pt（紧凑） |
| **标签** | 简短，优先使用单字标题 |

**推荐 tabs：**
1. **照片** - 照片图标
2. **回收站** - 垃圾桶图标
3. **设置** - 齿轮图标

### 1.2 照片网格

| 方面 | 指南 |
|------|------|
| **布局** | 使用 3 列网格 |
| **网格列** | 3 列典型（可根据尺寸类别为 2-4） |
| **间距** | 单元格间 2pt（iOS 照片使用 2pt） |
| **宽高比** | 正方形缩略图，`aspectFill` |
| **选择** | 使用视觉反馈（勾选标记 + 透明度） |

### 1.3 进度指示器

| 组件 | 何时使用 | 规格 |
|------|----------|------|
| **活动指示器**（旋转） | 持续时间未知 | 20pt（小），44pt（大）；内容居中 |
| **进度条** | 确定进度（0-100%） | 2-4pt 高度；从左到右填充 |
| **进度视图**（圆形） | 紧凑空间 | 用于受限区域 |

**扫描进度最佳实践：**
- 显示 "识别中 X/Y 项目" 和数字进度
- 对扫描操作使用确定进度条
- 内联放置或作为内容区域覆盖层
- 最小动画持续时间：0.3s 以保证可见性

### 1.4 设置 UI 模式

| 元素 | 模式 |
|------|------|
| **表格样式** | 分组样式（iOS 13+） |
| **切换单元格** | `UISwitch` 作为右侧附件视图 |
| **滑块单元格** | 最小/最大标签嵌入单元格内容 |
| **分段标题** | 将相关选项分组 |
| **导航** | 通过 `UINavigationController` 深入 |

**扫描范围滑块规格：**
- 离散值：1、2、3、6、12 个月
- 显示当前选择为文本标签
- 对离散值使用吸附行为

## 2. 配色方案推荐

### 2.1 系统颜色（适配深浅模式）

| 语义颜色 | 浅色模式 | 深色模式 | 用途 |
|----------|----------|----------|------|
| `systemBackground` | 纯白 | 纯黑 | 主背景 |
| `secondarySystemBackground` | 浅灰 | 深灰 | 分组表格背景 |
| `tertiarySystemBackground` | 白色 | 提升的深色 | 输入字段、卡片 |
| `systemFill` | 20% 黑色 | 30% 白色 | 微妙填充 |

### 2.2 语义强调色

| 颜色 | 用例 |
|------|------|
| `systemBlue` | 主要操作、选中状态、链接 |
| `systemGreen` | 成功、完成、正面操作 |
| `systemOrange` | 警告、需要注意 |
| `systemRed` | 破坏性操作（删除）、错误 |
| `systemGray` | 禁用状态、占位符 |

### 2.3 推荐配色方案

**主色：**
- **品牌蓝**: `systemBlue`（适配深浅）
- **成功绿**: `systemGreen` - 用于清理/完成状态
- **警告橙**: `systemOrange` - 用于中等风险项
- **危险红**: `systemRed` - 用于高风险/破坏性操作

**背景色：**
- 浅色: `systemBackground`（白色）/ `secondarySystemBackground`（灰色）
- 深色: `systemBackground`（黑色）/ `secondarySystemBackground`（深灰）

**文字颜色：**
- 主要: `label`（适配：浅色黑色，深色白色）
- 次要: `secondaryLabel`（60% 不透明度）
- 第三: `tertiaryLabel`（30% 不透明度）

## 3. 排版指南

### 3.1 系统字体层次结构

| 文字样式 | 默认大小 | 字重 | 用途 |
|----------|----------|------|------|
| `.largeTitle` | 34pt | 常规 | 标题栏中的应用名称 |
| `.title1` | 28pt | 常规 | 屏幕标题 |
| `.title2` | 22pt | 常规 | 分段标题 |
| `.title3` | 20pt | 常规 | 卡片标题 |
| `.headline` | 17pt | 半粗 | 强调内容 |
| `.body` | 17pt | 常规 | 主要内容 |
| `.callout` | 16pt | 常规 | 补充信息 |
| `.subheadline` | 15pt | 常规 | 介绍性文字 |
| `.footnote` | 13pt | 常规 | 说明、元数据 |
| `.caption1` | 12pt | 常规 | 图像说明 |
| `.caption2` | 11pt | 常规 | 小标签 |

### 3.2 字重推荐

| 字重 | 值 | 用例 |
|------|-----|------|
| 常规 | 400 | 正文、描述 |
| 中等 | 500 | 强调的 UI 元素 |
| 半粗 | 600 | 标题、选中状态 |
| 粗体 | 700 | 关键信息、应用名称 |

### 3.3 中文排版调整

| 方面 | 拉丁文字 | 中文 |
|------|----------|------|
| **最小大小** | 10pt | 11pt |
| **正文大小** | 17pt | 17pt（相同） |
| **行高** | 1.2-1.4× | 1.5-1.6× |
| **最小字重** | 任意 | 避免极细（字重 100-200） |

**布局考虑：**
- **文字扩展**: 中文可能比英文短 20-30%
- **换行**: 使用 `.byWordWrapping` 和 `.languageAware` 策略
- **字符间距**: 使用默认字距；避免紧密字母间距
- **垂直节奏**: 增加中文内容的段落间距

**翻译最佳实践：**
- Tab 标签保持 2-4 字符（照片、清理、设置）
- 使用简洁、动作导向的按钮文字
- 避免音译；使用清晰的中文术语

## 4. 触摸目标和间距

| 元素 | 最小大小 | 推荐 |
|------|----------|------|
| **标准按钮** | 44×44 pt | 48×48 pt |
| **Tab bar 项** | 25×25 pt 图标 | 44×44 pt 触摸目标 |
| **网格单元格** | 44×44 pt | 完整单元格大小 |
| **滑块拇指** | 28×28 pt | 32×32 pt |
| **切换开关** | 51×31 pt（系统） | 系统默认 |
| **导航按钮** | 44×44 pt | 48×48 pt |

### 4.1 间距指南

| 间距 | 用途 |
|------|------|
| **4pt** | 最小内边距，紧密分组 |
| **8pt** | 标准内部内边距 |
| **12pt** | 卡片内容内边距 |
| **16pt** | 屏幕边缘边距（iPhone） |
| **20pt** | 分段间距 |
| **24pt** | 大分段分隔 |

### 4.2 安全区域边距

- iPhone X+: 尊重安全区域内边距（顶部 44pt，底部 34pt）
- 横屏：对文字使用可读内容指南
- iPad：使用可读宽度约束居中内容

## 5. 动画规格

### 5.1 时间曲线

| 曲线 | 用途 | iOS API |
|------|------|---------|
| **Ease In Out** | 标准过渡 | `.curveEaseInOut` |
| **Ease Out** | 元素进入屏幕 | `.curveEaseOut` |
| **Ease In** | 元素离开屏幕 | `.curveEaseIn` |
| **Spring** | 交互，物理感觉 | `.spring()`（SwiftUI） |

### 5.2 持续时间指南

| 动画类型 | 持续时间 |
|----------|----------|
| **微交互**（按钮点击） | 0.1-0.15s |
| **标准过渡** | 0.3-0.4s |
| **复杂过渡** | 0.5-0.7s |
| **进度动画** | 连续（0.3s 循环） |

### 5.3 弹簧动画参数

```swift
// SwiftUI 弹簧规格
.withAnimation(.spring(
    response: 0.35,      // 持续时间
    dampingFraction: 0.7 // 弹跳（0-1）
))
```

### 5.4 推荐动画

| 交互 | 动画 |
|------|------|
| **Tab 切换** | 0.3s 交叉淡入淡出 |
| **照片选择** | 0.15s 缩放 + 不透明度 |
| **扫描进度** | 连续线性进度 |
| **卡片出现** | 0.4s 从底部弹簧 |
| **重复项间滑动** | 0.3s 滑动带弹簧 |

## 6. 无障碍考虑

### 6.1 最低要求

| 特性 | 规格 |
|------|------|
| **触摸目标** | 最小 44×44 pt |
| **对比度** | 正常文字 4.5:1，增强 7:1 |
| **大文字** | 支持动态类型（`.body` 及以上） |
| **旁白** | 所有交互元素有标签 |

### 6.2 实现清单

- [ ] 对所有文字支持动态类型
- [ ] 启用减少动画时测试
- [ ] 确保深色模式有足够颜色对比度
- [ ] 为照片网格项添加 `accessibilityLabel`
- [ ] 为操作按钮提供 `accessibilityHint`
- [ ] 支持语音控制命令

### 6.3 减少动画支持

```swift
// 检查减少动画偏好
if UIAccessibility.isReduceMotionEnabled {
    // 使用淡入淡出过渡代替动作
} else {
    // 使用完整弹簧动画
}
```

## 7. 组件规范

### 7.1 TabBar 组件

自定义 iOS 风格底部 tab bar：
- 3 个 tabs：照片、回收站、设置
- 激活/未激活图标状态
- 徽章支持计数指示器
- 模糊背景效果（iOS 13+ 风格）
- 安全区域处理

```typescript
interface TabBarProps {
  state: BottomTabNavigationState;
  descriptors: BottomTabDescriptorMap;
  navigation: NavigationHelpers;
  theme: AppThemePalette;
  photosCount: number;
  recycleBinCount: number;
}
```

### 7.2 PhotoGrid 组件

iOS 照片应用风格网格：
- 3 列布局（可配置）
- 按日期分段标题（类似 iOS 照片 "最近"）
- 选择模式带勾选覆盖
- 下拉刷新重新扫描
- 使用 expo-image 快速图像加载
- 虚拟化渲染提升性能

```typescript
interface PhotoGridProps {
  candidates: CleanupCandidate[];
  selectedIds: string[];
  onSelect: (id: string) => void;
  onItemPress: (candidate: CleanupCandidate) => void;
  onLoadMore?: () => void;
  refreshing: boolean;
  onRefresh: () => void;
  theme: AppThemePalette;
  language: AppLanguage;
  filter: RecognitionFilter;
}
```

### 7.3 ScanProgress 组件

扫描期间全屏覆盖：
- 动画圆形进度
- 滚动数字动画的计数器
- 当前文件名/数字显示
- 取消按钮
- 背景模糊

```typescript
interface ScanProgressProps {
  visible: boolean;
  scannedCount: number;
  totalCount: number;
  currentAssetName?: string;
  onCancel: () => void;
  theme: AppThemePalette;
}
```

### 7.4 ScanCounter 组件

动画数字计数器：
- 滚动数字动画（老虎机风格）
- 数字间平滑过渡
- 可配置持续时间

## 8. 总结：误触清理的关键推荐

1. **Tab Bar**: 底部定位，3 个 tabs（照片、回收站、设置），SF Symbols 图标，44pt 触摸目标
2. **照片网格**: 3 列布局，2pt 间距，正方形缩略图，选择勾选标记带透明度反馈
3. **进度**: 确定进度条带 "识别中 X/Y 项目" 标签，0.3s 动画循环
4. **设置**: 分组表格样式，扫描范围离散滑块
5. **颜色**: 使用系统语义颜色自动适配深浅模式
6. **排版**: 17pt 正文，34pt 应用标题，支持动态类型，中文 1.5× 行高
7. **动画**: 交互 0.3-0.4s 弹簧动画，尊重减少动画
8. **无障碍**: 44pt 最小触摸目标，4.5:1 对比度，旁白标签
